HTML

Hypertext Markup Language (HTML) is the standard markup langage for creating web pages and web applications. web vrewsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other object, such as ineractive forms, may be embedded into the renderred page. It provides a means to create structured document by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
